API Documentation: Send Single WhatsApp Message

Endpoint

GET https://api.whatsappros.in/api/single/message

Description

This API sends a single WhatsApp message to the specified number using an authentication token. The request must contain a valid token, number, and message as query parameters.

Query Parameters

Parameter Type Description Required
token string A valid base64-encoded token used for authentication Yes
number string The WhatsApp phone number in international format without the `+` Yes
message string The message to send Yes

Example Request

GET https://api.whatsappros.in/api/single/message?token=eyJpcCI6IjE2OC42NiMi4yIiwidGsiOiJJSTM0SU0ifQ==&number=93******97&message=hello

Success Response

Status: 200 OK

Response:
  success: true
  response:
    id: "message-id"
    ack: "1"
    message: "Message sent successfully"
    number: "93******97"
    timestamp: "2024-09-10T12:45:30.000Z"

Error Responses

Invalid Token

Status: 400 Bad Request

Response:
  message: "Invalid token"

Missing Parameters

Status: 400 Bad Request

Response:
  message: "Number cannot be blank"

Internal Server Error

Status: 500 Internal Server Error

Response:
  message: "Internal Server Error"
